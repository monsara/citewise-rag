# Sample Documents

Ця папка містить тестові документи для розробки та тестування системи.

## Формати

Підтримувані формати для MVP:
- PDF
- TXT
- Markdown (.md)

## Рекомендації для тестових документів

1. **Розмір**: 1-50 сторінок для початкового тестування
2. **Контент**: різноманітний за темами для перевірки релевантності
3. **Структура**: з заголовками, списками, форматуванням
4. **Мова**: англійська та українська для тестування мультимовності

## Приклади документів

- Технічну документацію
- Статті
- Навчальні матеріали
- FAQ документи
